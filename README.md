Tools to have
==============

___________
|	  |
| lolcat  |
|_________|
	
	on Arch
	========
		sudo pacman -S lolcat

	on Fedora
	==========
		sudo dnf install lolcat

	on Opensuse
	============
		sudo zypper install lolcat

	on Debian(Ubuntu)
	=================
		sudo apt-get install lolcat
